## 🚀 How to Upload

### Option 1: Upload All Files at Once (Easiest)

1. Go to: **github.com/allenmanoba/cornerstone-photography**
2. Click **"Add file" → "Upload files"**
3. Drag ALL files from your outputs folder into the upload box (including the css and js folders)
4. At the bottom, click **"Commit changes"**
5. Wait 1-2 minutes for deployment

### Option 2: Upload Manually via Web Interface

1. For each file/folder:
   - Click **"Add file" → "Create new file"**
   - Name the file (e.g., `services.html` or `css/styles.css`)
   - Copy-paste the content from each file
   - Click **"Commit changes"**

## ✨ What Happens Next
